## 一、系统概述

### 1.1 系统简介

**极光法律服务管理系统**是一套专为律师事务所和企业客户设计的法律服务全流程管理平台。系统以"不做事后救火员，要做全程守护者"为核心理念，实现：

- ✅ **合同风险早发现、早预防** - 律师在合同签订前全面审查
- ✅ **履约过程有记录、有证据** - 全流程跟踪，自动固化证据链
- ✅ **发生争议有准备、有底气** - 律师持续跟进，一旦出现争议立即响应
- ✅ **诉讼启动有材料、有优势** - 系统一键导出完整诉讼材料

### 1.2 四大核心模块

| 模块 | 功能概要 | 主要价值 |
|------|---------|---------|
| **合同管理** | 合同审查、履约跟踪、对账提醒、证据固化 | 全周期管控，风险前置化解 |
| **案件管理** | 案情跟踪、开庭提醒、材料归档、进度透明 | 实时掌握案情，全程透明可追溯 |
| **重大事项研讨** | 问题建档、@律师响应、文件共享、方案讨论 | 专业团队协作，快速解决复杂问题 |
| **数据分析中心** | 多维度看板、智能预警、统计报表、数据导出 | 科学决策，精准把控法律风险 |

### 1.3 系统价值

- 🎯 **省心** - 律师全程陪跑，合同履行不再担心
- ⚡ **省时** - 证据自动固化，争议发生即刻应对
- 💰 **省钱** - 沟通成本骤降，律师费用大幅优化
- 🚀 **省力** - 系统一键导出，诉讼材料自动生成

## 二、快速上手

### 2.1 首次登录

#### 访问地址
- 系统首页：打开浏览器访问系统地址
- 点击右上角"登录系统"按钮

#### 登录凭证
根据您的用户角色，使用不同的登录入口：

**企业用户登录**：
- 手机号：您的注册手机号
- 密码：由律师团队提供的初始密码（首次登录后建议修改）

**律师用户登录**：
- 手机号：您的律师账号手机号
- 密码：由管理员分配的初始密码

### 2.2 修改密码

首次登录后，强烈建议立即修改密码：

1. 点击右上角头像 → 选择"个人资料"
2. 在个人资料页面，找到"修改密码"选项
3. 输入当前密码、新密码、确认新密码
4. 点击"保存"完成修改

💡 **密码要求**：至少6个字符，建议包含字母和数字

### 2.3 界面导航

登录后，系统主要导航区域：

- **顶部导航栏**：
  - 系统Logo（点击返回首页）
  - 主要功能菜单（合同、案件、重大事项、数据分析等）
  - 搜索框（快速检索）
  - 待办提醒（小红点提示）
  - 用户头像菜单

- **首页工作台**（企业用户）：
  - 待办事项汇总
  - 紧急提醒
  - 到期提醒
  - 公告通知

- **首页工作台**（律师用户）：
  - 待办事项汇总
  - 企业客户列表
  - 团队工作量统计
  - 公告通知

### 2.4 基本操作流程

#### 企业用户典型流程
```
1. 登录系统 → 查看工作台待办
2. 新建合同 → 上传合同文件 → 等待律师审查
3. 查看律师评论 → 修改合同 → 标记为已处理
4. 定期查看对账提醒 → 更新履约记录
5. 发生争议 → 创建重大事项 → @律师讨论
6. 查看数据分析 → 导出报表
```

#### 律师用户典型流程
```
1. 登录系统 → 查看待办事项
2. 进入企业 → 查看待审查合同
3. 审查合同 → 添加评论意见 → 标记为已审查
4. 跟进案件 → 更新案情进度 → 上传材料
5. 响应重大事项 → 提供专业意见
6. 查看数据分析 → 优化团队工作分配
```

## 三、用户角色说明

### 3.1 企业用户角色

系统为企业用户设置了三种角色，权限递增：

#### 👤 员工（Employee）
- **权限范围**：
  - ✅ 查看所属企业的合同、案件、重大事项
  - ✅ 创建合同、重大事项
  - ✅ 添加评论、上传附件
  - ❌ 删除合同、案件、附件
  - ❌ 查看数据分析报表

- **适用场景**：普通员工、合同经办人

#### 🎖️ 高管（Executive）
- **权限范围**：
  - ✅ 员工的所有权限
  - ✅ 删除附件
  - ✅ 查看数据分析报表
  - ✅ 导出数据报表
  - ❌ 删除合同、案件

- **适用场景**：部门经理、财务总监、法务主管

#### 👑 企业主（Boss）
- **权限范围**：
  - ✅ 所有权限
  - ✅ 删除合同、案件、重大事项
  - ✅ 确认员工的删除请求
  - ✅ 管理企业用户账号

- **适用场景**：企业法定代表人、董事长、总经理

### 3.2 律师用户角色

系统为律师事务所设置了五种角色：

#### 📋 律师助理（Assistant）
- **权限范围**：
  - ✅ 查看分配的案件
  - ✅ 更新案件进度
  - ✅ 添加工作日志
  - ❌ 审查合同
  - ❌ 标记为已审查

- **适用场景**：实习律师、律师助理

#### ⚖️ 律师（Lawyer）
- **权限范围**：
  - ✅ 律师助理的所有权限
  - ✅ 审查合同并添加意见
  - ✅ 标记合同为已审查
  - ✅ 创建案件、分配团队成员
  - ✅ 响应重大事项

- **适用场景**：执业律师

#### 🌟 资深律师（Senior Lawyer）
- **权限范围**：
  - ✅ 律师的所有权限
  - ✅ 查看团队工作量统计
  - ✅ 指导团队成员

- **适用场景**：资深律师、合伙人律师

#### 👥 团队负责人（Team Leader）
- **权限范围**：
  - ✅ 资深律师的所有权限
  - ✅ 管理团队成员
  - ✅ 分配企业客户给团队
  - ✅ 查看团队所有数据

- **适用场景**：律师团队负责人

#### 🔑 超级管理员（Super Admin）
- **权限范围**：
  - ✅ 所有权限
  - ✅ 创建企业客户
  - ✅ 管理律师账号
  - ✅ 查看所有企业数据

- **适用场景**：律师事务所管理合伙人

### 3.3 管理员角色

系统后台管理员（仅用于系统维护）：

#### 🛡️ 管理员（Admin）
- **权限范围**：系统配置、用户管理、公告发布

#### 🛡️ 超级管理员（Super Admin）
- **权限范围**：所有管理员权限 + 管理员账号管理

## 四、核心功能模块

### 4.1 合同管理

#### 📝 创建合同

**操作路径**：导航栏 → 合同管理 → 新建合同

**操作步骤**：

1. **选择创建模式**：
   - **快速模式**：仅填写核心字段（合同名称、对方企业、金额、日期）
   - **完整模式**：填写所有详细信息

2. **填写基本信息**：
   - 合同名称（必填）
   - 对方企业名称（必填）
   - 合同类型（选填）
   - 合同金额（必填）
   - 签订日期、开始日期、到期日期（必填）

3. **上传合同文件**：
   - 点击"上传合同文件"按钮
   - 支持格式：PDF、Word、图片
   - 可上传多个文件

4. **填写合同方信息**（可选）：
   - 我方签约主体
   - 对方签约主体
   - 对方联系人、联系电话

5. **设置对账周期**（可选）：
   - 对账周期天数（默认30天）
   - 系统将按周期自动提醒对账

6. **点击"创建合同"完成**

💡 **小贴士**：
- 首次创建建议使用快速模式，后续可在详情页补充完整信息
- 合同文件支持多个，建议将合同正文、附件分别上传
- 设置对账周期后，系统会自动提醒，避免遗漏对账

#### 🔍 查看合同详情

**操作路径**：合同列表 → 点击合同名称

**详情页包含**：

**基础信息区**：
- 合同名称、类型、状态
- 签约双方信息
- 合同金额、日期范围
- 合同文件（点击预览或下载）

**律师审查区**（仅律师可见）：
- 审查状态（待审查/已审查）
- 审查律师、审查时间
- 审查意见（在评论区查看）

**履约跟踪区**：
- 履约进度条
- 关键节点记录
- 交付日期、验收日期
- 质保期到期提醒

**对账单区**：
- 对账周期设置
- 历史对账记录
- 快速创建本月对账单
- 对账单评论记录

**风险管理区**：
- 风险等级（低/中/高）
- 争议发生情况
- 相关案件链接

**评论讨论区**：
- 查看所有评论（合同评论 + 对账单评论）
- 添加新评论
- @提及律师或企业用户
- 上传附件

**操作按钮区**：
- 编辑合同信息
- 标记为已审查（仅律师）
- 续签合同
- 导出归档文件
- 删除合同（需密码确认）

#### 📊 合同列表筛选

**筛选条件**：
- **搜索框**：按合同名称搜索
- **企业筛选**（仅律师）：选择特定企业
- **标签筛选**：按合同标签分类
- **状态筛选**：草稿/执行中/已完成/已违约/诉讼中
- **日期范围**：按签订日期、到期日期筛选

**排序方式**：
- 创建时间（默认）
- 签订日期
- 到期日期
- 合同金额

#### 📅 合同日历视图

**操作路径**：导航栏 → 合同管理 → 日历视图

**功能说明**：
- 按月份展示合同关键日期
- 支持事件类型筛选：
  - 签订日期
  - 到期日期
  - 履行开始/结束
  - 交付/验收日期
  - 质保到期
  - 下次跟进
  - 争议发生/结案
  - 最后续签

- 点击日期事件可快速跳转到合同详情

#### 📂 对账单管理

**创建对账单**：
1. 进入合同详情页
2. 找到"对账单"区域
3. 点击"快速创建本月对账单"
4. 填写对账信息：
   - 对账周期（自动带出）
   - 应收/应付金额
   - 实收/实付金额
   - 对账备注
5. 点击"创建"完成

**对账单评论**：
- 支持在对账单下添加评论
- 评论会同步显示在合同评论区
- @律师可获得即时提醒

#### 🏷️ 合同标签

**添加标签**（仅律师）：
1. 进入合同详情页
2. 点击"添加标签"按钮
3. 选择现有标签或创建新标签
4. 点击"保存"

**标签用途**：
- 快速分类：采购合同、销售合同、服务合同等
- 快速筛选：按标签批量查看合同
- 智能归档：系统推荐相似合同

### 4.2 案件管理

#### ⚖️ 创建案件

**操作路径**：导航栏 → 案件管理 → 新建案件

**操作步骤**：

1. **填写基本信息**：
   - 案件名称（必填）
   - 案号（选填，立案后填写）
   - 案件类型（民事/刑事/行政/执行/仲裁）
   - 案件状态（准备立案/已立案待审/审理中等）
   - 案件阶段（仲裁/一审/二审/执行等）
   - 诉讼地位（原告/被告/第三人）

2. **填写当事人信息**：
   - 我方当事人（原告/被告）
   - 对方当事人（被告/原告）
   - 第三人（如有）
   - 支持添加多个当事人

3. **填写诉讼请求**：
   - 诉讼标的金额（必填）
   - 详细诉讼请求
   - 案情摘要

4. **填写关键日期**：
   - 立案日期
   - 开庭时间
   - 判决领取日期
   - 归档日期
   - 上诉期限

5. **设置案件团队**：
   - 主办律师（必填）
   - 协办律师（可多选）
   - 律师助理（可多选）

6. **上传案件材料**：
   - 起诉状
   - 证据材料
   - 开庭笔录
   - 判决书
   - 其他文件
   - 支持按审理阶段分类上传

7. **点击"创建案件"完成**

💡 **小贴士**：
- 案件可从合同快速创建，自动关联合同信息
- 开庭时间会自动加入系统提醒
- 案件材料支持分阶段管理（立案材料、庭审材料、判决材料等）

#### 📋 查看案件详情

**操作路径**：案件列表 → 点击案件名称

**详情页包含**：

**基础信息区**：
- 案件名称、案号、状态、阶段
- 案件类型、诉讼地位
- 关联合同（如有）
- 当事人信息

**诉讼请求区**：
- 诉讼标的金额
- 详细诉讼请求
- 案情摘要

**案件团队区**：
- 主办律师
- 协办律师
- 律师助理
- 支持添加/移除团队成员

**关键日期区**：
- 立案日期
- 开庭时间（会自动提醒）
- 判决领取日期
- 归档日期
- 上诉期限
- 执行到期日

**材料文件区**（按阶段分类）：
- 立案材料
- 开庭材料
- 判决书
- 归档材料
- 支持预览、下载、删除

**律师费管理区**：
- 律师费金额
- 付款状态（待付款/部分付款/已付清）
- 付款记录
- 开票状态

**财产保全区**（如有）：
- 保全类型
- 保全金额
- 保全到期日

**工作日志区**：
- 查看所有工作日志
- 按时间倒序排列
- 支持添加附件

**评论讨论区**：
- 查看所有评论
- 添加新评论
- @提及团队成员

**关联案件区**：
- 关联案件列表
- 支持添加/移除关联

**操作按钮区**：
- 编辑案件信息
- 更新财产保全
- 更新律师费
- 添加工作日志
- 导出案件归档
- 导出所有材料
- 删除案件（需密码确认）

#### 📊 案件列表筛选

**筛选条件**：
- **搜索框**：按案件名称、案号搜索
- **企业筛选**（仅律师）：选择特定企业
- **状态筛选**：准备立案/已立案待审/审理中/已判决/执行中/调解结案/已归档
- **阶段筛选**：仲裁/一审/二审/执行/再审/恢复执行
- **案件类型**：按案件类型筛选
- **诉讼地位**：原告/被告/第三人
- **团队成员**：按主办律师或协办律师筛选
- **日期范围**：按立案日期筛选
- **优先级**：低/中/高/紧急
- **财产保全**：有财产保全/无财产保全

**快捷筛选**：
- 我的案件：显示当前律师参与的所有案件
- 主办案件：显示当前律师作为主办律师的案件
- 即将开庭：显示7天内要开庭的案件
- 判决待领：显示判决已作出但未领取的案件

#### 📅 案件日历视图

**操作路径**：导航栏 → 案件管理 → 日历视图

**功能说明**：
- 按月份展示案件关键日期
- 支持事件类型筛选：
  - 立案日期
  - 开庭时间（红色高亮）
  - 判决领取（黄色提示）
  - 归档日期
  - 保全到期（橙色预警）
  - 执行到期（红色预警）

- 点击日期事件可快速跳转到案件详情

#### 📝 工作日志

**添加工作日志**：
1. 进入案件详情页
2. 找到"工作日志"标签页
3. 点击"添加工作日志"
4. 填写日志信息：
   - 日志标题（必填）
   - 工作内容（必填）
   - 工作日期（默认今天）
   - 提交人（自动填充）
   - 上传附件（可选）
5. 点击"保存"完成

**查看工作日志**：
- 按时间倒序排列
- 显示提交人、提交时间
- 支持查看附件
- 支持编辑、删除（仅提交人）

#### 👥 案件团队协作

**添加团队成员**：
1. 进入案件详情页
2. 找到"案件团队"区域

## 六、常见问题解答

### 6.1 登录相关

**Q1：忘记密码怎么办？**

A：请联系律师团队或系统管理员重置密码。暂不支持自助找回密码。

**Q2：手机号被锁定怎么办？**

A：系统设置了防暴力破解机制，连续输错密码5次后账号会被锁定1小时。如需立即解锁，请联系管理员。解锁链接会发送到您的手机号码。

**Q3：多人可以使用同一账号吗？**

A：不建议多人共用账号。系统会记录所有操作日志，共用账号会导致操作记录混乱，影响责任追溯。建议每人单独注册账号。

### 6.2 合同管理

**Q4：合同上传后律师多久会审查？**

A：通常情况下，律师会在24小时内完成审查。如果合同比较复杂，可能需要更长时间。如有紧急情况，可通过重大事项@律师加急处理。

**Q5：合同可以删除吗？**

A：可以。企业主和律师有权删除合同。删除时需要输入密码确认，删除后无法恢复，请谨慎操作。建议仅在录入错误时删除，已归档合同不建议删除。

**Q6：如何设置合同对账提醒？**

A：创建或编辑合同时，设置"对账周期天数"（默认30天）。系统会自动按周期提醒对账。您也可以在合同详情页手动创建对账单。

**Q7：合同文件支持哪些格式？**

A：支持PDF、Word（.doc/.docx）、Excel（.xls/.xlsx）、图片（JPG/PNG）、压缩包（ZIP/RAR）等常见格式。单个文件最大50MB。

### 6.3 案件管理

**Q8：案件材料如何分类管理？**

A：系统按审理阶段自动分类材料：
- 立案材料：起诉状、证据材料等
- 开庭材料：开庭笔录、质证意见等
- 判决材料：判决书、裁定书等
- 归档材料：结案后的所有材料

上传时可以选择材料类型，也可以在详情页按阶段分类查看。

**Q9：如何导出案件归档？**

A：进入案件详情页，点击"导出案件归档"按钮。系统会生成ZIP压缩包，包含：
- 案件基本信息PDF
- 所有上传的材料文件
- 工作日志汇总

**Q10：案件团队成员有什么权限？**

A：
- 主办律师：所有权限（查看、编辑、删除、分配任务）
- 协办律师：查看、编辑案件信息、添加工作日志
- 律师助理：查看案件信息、添加工作日志

### 6.4 重大事项

**Q11：重大事项和评论有什么区别？**

A：
- **评论**：针对某个合同或案件的具体问题，直接在详情页讨论
- **重大事项**：独立的法律问题，需要建档研讨，可能涉及多个合同或案件

简单问题用评论，复杂问题建重大事项。

**Q12：律师多久会答复重大事项？**

A：通常情况下，律师会在48小时内作出答复。紧急事项会优先处理。如果超过48小时未答复，系统会发送催办提醒。

**Q13：如何关注重大事项？**

A：进入重大事项详情页，点击"关注"按钮。关注后，该事项的任何更新（新评论、状态变化）都会推送通知给您。

### 6.5 数据分析

**Q14：数据分析报表可以导出吗？**

A：可以。进入数据分析页面，点击"导出CSV报表"按钮，即可下载Excel格式的数据报表。报表包含所有统计数据和明细信息。

**Q15：律师费分析中的筛选条件是什么意思？**

A：
- **时间维度**：
  - 立案日期：按案件立案时间统计
  - 回款日期：按律师费实际回款时间统计
  - 开票日期：按发票开具时间统计
  - 结案日期：按案件结案时间统计

- **付款状态**：
  - 待付款：尚未支付律师费
  - 部分付款：已支付部分律师费
  - 已付清：律师费已全部支付

- **开票状态**：
  - 已开票：已开具发票
  - 未开票：尚未开具发票

**Q16：企业用户可以看到律师费分析吗？**

A：企业用户可以查看自己企业的律师费统计（案件总数、律师费总额、付款情况）。但无法查看其他企业的数据，也无法查看律师工作量统计。

### 6.6 文件和附件

**Q17：上传的文件可以删除吗？**

A：
- 企业用户：仅高管和企业主可以删除附件
- 律师用户：可以删除自己上传的附件

删除前请确认，删除后无法恢复。

**Q18：文件预览不了怎么办？**

A：
- 确认文件格式是否支持预览（PDF、图片支持，Word/Excel部分支持）
- 尝试下载到本地后用对应软件打开
- 如果文件损坏，请重新上传

**Q19：可以批量下载文件吗？**

A：可以。在合同或案件详情页，点击"导出归档"按钮，系统会将所有文件打包成ZIP压缩包供下载。

### 6.7 权限和安全

**Q20：如何确保数据安全？**

A：系统采取多重安全措施：
- 所有数据传输采用HTTPS加密
- 文件存储采用云端加密存储
- 严格的权限控制，用户只能访问自己权限范围内的数据
- 完整的操作日志记录，可追溯所有操作
- 定期数据备份，防止数据丢失

**Q21：企业员工可以看到所有合同吗？**

A：企业员工可以查看所属企业的所有合同、案件、重大事项。但无法查看其他企业的数据。

**Q22：律师可以看到所有企业的数据吗？**

A：律师采用团队权限控制：
- 只能查看分配给自己团队的企业数据
- 超级管理员可以查看所有企业数据
- 团队负责人可以查看团队所有数据

### 6.8 系统使用

**Q23：系统支持手机访问吗？**

A：系统采用响应式设计，支持手机、平板访问。建议使用Chrome、Safari等现代浏览器访问，以获得最佳体验。

**Q24：系统对浏览器有要求吗？**

A：建议使用以下浏览器：
- Chrome 80+
- Firefox 75+
- Safari 13+
- Edge 80+
- Opera 67+

避免使用IE浏览器，可能会出现兼容性问题。

**Q25：如何联系技术支持？**

A：
- 系统使用问题：联系律师团队或系统管理员
- 技术故障：通过系统内"帮助与反馈"提交工单
- 紧急情况：拨打客服电话（由律师团队提供）
