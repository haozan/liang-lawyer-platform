## 🚀 Phase 1.1：优化合同表单（优先）

### 目标
- 创建"快速创建"模式，录入时间从15分钟缩短到3分钟
- 保留"完整创建"模式，满足详细录入需求
- 优化用户入口，清晰引导用户选择

### 实施内容

#### 1. 创建快速创建表单
文件：`app/views/contracts/_quick_form.html.erb`
- 只包含10个核心必填字段
- 单页表单，无标签页
- 优化字段布局和提示

#### 2. 修改控制器逻辑
文件：`app/controllers/contracts_controller.rb`
- 支持mode参数（quick/full）
- 快速模式只验证必填字段
- 完整模式保持原有验证

#### 3. 更新入口页面
文件：`app/views/contracts/index.html.erb`
- 添加两个创建按钮（快速/完整）
- 推荐快速创建模式
- 明确两种模式的区别

#### 4. 优化新建页面
文件：`app/views/contracts/new.html.erb`
- 根据mode参数渲染不同表单
- 提供模式切换功能
- 优化页面布局

### 预期效果
- ✅ 快速模式：3分钟完成录入
- ✅ 完整模式：保留原有功能
- ✅ 用户放弃率降低50%
